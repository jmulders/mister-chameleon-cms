{{# collection_entry.antlers.html — Headless preview bridge for collection entries. Works for any collection (blog, vacancies, case_studies, team_members, …). Uses {{ uri }} — the full path Statamic assigns via the collection route — rather than {{ slug }} alone, so the redirect always points to the correct Next.js URL regardless of route nesting. ─── Two contexts ──────────────────────────────────────────────────────────── 1. Direct browser visit (window.self === window.top) Redirect immediately to the corresponding Next.js page. 2. CP Live Preview iframe (window.self !== window.top) Load the published Next.js page inside an iframe so the editor can see a live preview while editing the entry. ─── Note on draft previews ────────────────────────────────────────────────── For full draft-token support (showing unsaved changes in the preview), wire up a collection-specific draft API endpoint in Next.js and call it here instead of loading the published URL directly. #}}
